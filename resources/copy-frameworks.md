# Copy Frameworks

Reusable copy structures for common page types. Pass these to the copywriter agent as starting frameworks.

---

## RV Park / Campground

### Hero Copy Formula
`[Adjective] [Location] camping [benefit].` + CTA
Example: "Spacious riverside camping in the heart of the Ozarks. Book your site today."

### Amenities Section Formula
Lead with the most differentiating amenity. Follow with supporting list. Close with reassurance ("pet friendly", "open year-round").

### Rates Page Formula
Intro line → site type table → seasonal notes → booking CTA.

---

## Housing / Real Estate

### Hero Copy Formula
`[Community name] — [key benefit] in [location].` + CTA
Example: "Maplewood Estates — affordable new construction in Cedar Rapids. View available homes."

### Listings Page Formula
Filter bar intro → listing grid → "Can't find what you're looking for?" CTA to contact.
